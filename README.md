# DanChess Game ♟️

**Учебный проект**, созданный в процессе изучения программирования. Я решил начать с шахмат, так как это отличный способ попрактиковаться в логике и алгоритмах.

> **Статус проекта:** В разработке (MVP). Игра работает полностью в консоли/терминале.

## 🎯 Почему этот проект важен для меня?
Этот проект входит в мой список приоритетных идей для практики. Шахматы помогают мне отточить навыки:
* Проектирования логики движений фигур.
* Обработки правил игры (шах, мат, рокировка).
* Написания чистого и понятного кода.

## ✨ Текущие возможности
* Отображение доски в терминале.
* Логика ходов для большинства фигур.
* Поочередные ходы для двух игроков.
